What frameworks were used to build DogDubber.com?

DogDubber.com was created using the Django and Pytorch frameworks

How does the website recognize which breed the dog is?

Pytorch was used to tune a pre-trained mobilenet_v2 model. This model was then loaded into the website to classify the images uploaded into a dog breed

What are the shortcomings of this model?

Only 120 dog breeds can be recongized even though more breeds exist. Also, the model's accuracy varies based on the dog's position in the image.
